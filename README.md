## Getting Started
###### backend
1. `cd backend`
2. Install dependencies `npm install` or `yarn install`
3. Start the server `npm run start` or `yarn start`

###### frontend
1. `cd client`
2. `cd nextapp`
3. Install dependencies `npm install` or `yarn install`
3. Start the server `npm run dev` or `yarn dev`
